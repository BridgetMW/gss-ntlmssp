
BUILDING
========

Usual tools of the trade:

$ autoreconf -f -i
$ ./configure
$ make

optionally (for easy installation on Fedora):
$ make rpms


In order to build gss-ntlmssp the following development packages are needed:

    autoconf
    automake
    docbook-style-xsl
    doxygen
    findutils
    krb5-devel
    libtool
    libxml2
    libxslt
    libunistring-devel
    m4
    pkgconfig
    openssl-devel
